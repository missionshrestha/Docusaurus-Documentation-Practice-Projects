# Pricing

TaskForge offers flexible plans so you can start small and scale with your team.

## Plans

- **Free**
  - For individuals and small side projects.
  - Limited projects and API rate limits.
- **Pro – $19/user/month**
  - For growing teams that need more automation.
  - Higher limits, priority support.
- **Enterprise – Contact us**
  - For large organizations with advanced security and compliance needs.

## Feature comparison

| Feature                  | Free        | Pro                 | Enterprise                |
| ------------------------ | ---------- | ------------------- | ------------------------- |
| Projects                 | 3          | Unlimited           | Unlimited                 |
| Team members             | 5          | Unlimited           | Unlimited                 |
| API access               | Basic      | Full                | Full + custom endpoints   |
| CLI & automation         | Yes        | Yes                 | Yes                       |
| SSO & SAML               | No         | No                  | Yes                       |
| Dedicated support        | Community  | Email               | Dedicated manager         |
| Data residency options   | No         | Limited             | Region-specific           |
