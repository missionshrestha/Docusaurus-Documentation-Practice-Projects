This quickstart will help you:

1. Install the TaskForge CLI
2. Log in (optional for this fictional project)
3. Create your first project

## 1. Install the CLI

Make sure you have [Node.js](https://nodejs.org/) installed (version 18 or higher).

Then install the TaskForge CLI globally:

```bash
npm install -g taskforge
```

## 3. Next steps

Next, learn how to [create projects](guides/creating-projects.mdx).

You can also explore more CLI commands in the [CLI Usage](guides/cli-usage.md) guide.
